Introdução à Linguagem C

A linguagem C foi criada na década de 1970 por Dennis Ritchie. É uma linguagem estruturada, compilada e muito utilizada no desenvolvimento de sistemas operacionais, softwares embarcados e aplicações que exigem alto desempenho.

Variáveis e Tipos de Dados

Em C, as variáveis precisam ser declaradas antes de serem utilizadas.

  • int: números inteiros
  • float: números decimais
  • char: caracteres
  • double: números decimais com maior precisão

Entrada e Saída de Dados

Utilizamos printf() para saída e scanf() para entrada.

Estruturas Condicionais

  • if
  • else
  • else if
  • switch

Laços de Repetição

  • for
  • while
  • do while

Funções

A função principal de um programa em C é main().
